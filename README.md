# News App Flutter 

This is a News App mobile application built using Flutter/Dart. API used from newsapi.org

## Features

- Users can browse through various categories of news.
- Users can view the details of a particular news
- Users can search for news

## Installation

- Clone the repository to your local machine
- Open the project in your IDE
- Sync the project 
- Run the app on an emulator or a physical device
- Then, paste your api key here : <br> <img src="https://github.com/Egemendokkodo/news_app_flutter/assets/70278989/7b2c0093-ceb8-4f64-8115-e9ef6950b8ba" width="400" height="200">


## Technologies Used

- Flutter/Dart

- Shimmer
- API
